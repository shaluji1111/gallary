# Sourav's Wedding Gallery

A beautiful, password-protected wedding photo gallery hosted on Vercel with images on Cloudinary.

## Setup

1. Install dependencies: `npm install`
2. Copy manifest: `cp ../manifest.json ./public/manifest.json`
3. Run locally: `npm run dev`
4. Build: `npm run build`

## Environment Variables

```
NEXT_PUBLIC_GALLERY_PASSWORD=shalu
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=deftovszl
```

Set these in Vercel dashboard under Project → Settings → Environment Variables.
